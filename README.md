# Dungeon (Подземелье)

Игра dungeon (Подземелье) с графическим (ASCII) интерфейсом от первого лица. Исследуйте подземелья, сражайтесь с врагами, собирайте предметы и решайте головоломки.

## Описание

Игра представляет собой пошаговый roguelike, где вы видите мир от первого лица. Ваша задача — найти выход из подземелья, преодолевая препятствия: закрытые двери, ловушки, сейфы с кодами и агрессивных врагов.

## Управление

### Движение
- `w` — шаг вперёд
- `s` — шаг назад
- `a` — шаг влево
- `d` — шаг вправо
- `q` — повернуть налево
- `e` — повернуть направо
- `z` — развернуться

### Действия
- `f` — взаимодействовать (двери, рубильники, ящики, сейфы)
- `m` — посмотреть карту
- `i` — открыть инвентарь
- `j` — атаковать врага
- `g` — обыскать труп
- `k` — использовать аптечку
- `help` — список команд

## Элементы игры

| Символ | Объект | Описание |
|--------|--------|----------|
| `#` | Стена | Непроходимое препятствие |
| `.` | Проход | Пустая клетка |
| `_`, `¯`, `[`, `]` | Дверь/Ворота | Требуют ключ или рубильник |
| `🗝` | Ключ | Открывает двери с соответствующим номером |
| `📄` | Записка | Содержит подсказки или код от сейфа |
| `🧰` | Аптечка | Восстанавливает здоровье |
| `+`/`-` | Рубильник | Управляет воротами |
| `❒` | Ящик | Содержит предметы |
| `S` | Сейф | Требует код, внутри предметы |
| `E` | Выход | Цель уровня |
| `^`, `v`, `>`, `<` | Игрок | Направление взгляда |
| `†` | Труп | Можно обыскать |
| (ASCII модель) | Враг | Атакует при обнаружении |

## Инвентарь

- Максимум 5 предметов
- Аптечки восстанавливают здоровье
- Записки можно читать
- Ключи автоматически используются для открытия дверей

## Враги

- Патрулируют территорию
- Замечают игрока в поле зрения
- Преследуют и атакуют в ближнем бою
- После смерти оставляют труп с предметами

## Уровни

1. **Подвал Виктора Носаня** — обучающий уровень с сейфом
2. **Лабиринт ужасов** — лабиринт с врагами
3. **Школа №24** — уровень с подъёмными воротами
4. **Под СИНХом** — сложный уровень с множеством механизмов
